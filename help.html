Managing Parts

Adding Notes

The tab grid is divided into a collection of bars, which in turn are divided into crotchets (a single note / beat) and quavers (a quarter-note). Each horizontal line represents a string of the part's instrument.

Notes can be added to a part by clicking on a quaver on a string and typing the fret number. If a note already exists, it will be overwritten. If a fret number too large for the instrument has been typed, the last number typed will be entered.

Navigating the grid

The following keys can be used when the grid is selected:

  • Numeric - add a fret number
  • Delete - delete a fret number
  • Backspace - remove the last digit of a fret number
  • Cursor - to move up / down / left / right by 1 position
  • Ctrl-cursor - skip vertically to the first or last string, skip horizontally by 1 bar
  • Tab / shift-tab - skip forwards / backwards by 1 crotchet
  • Home - move to 1st quaver

Selecting a range

A range of notes can be selected by using the shift key when the grid is selected. Shift-click, shift-cursor or click-drag to highlight a range.

The following keys can be used when a range is selected:

  • Ctrl-c - copy the fret numbers
  • Ctrl-v - paste the copied fret numbers relative to the current position
  • Delete - delete the fret numbers
